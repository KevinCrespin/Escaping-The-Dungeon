# Escaping-The-Dungeon

A 3D top-down view roguelike video game made with Unity.

## Play it!

Download this repository and run the Escaping The Dungeon.exe file.

## Controls

    (Move Up)           W
    (Move Left)         A
    (Move Down)         S
    (Move Right)        D
    (Place Bomb)        Space
    (Pause)             ESC
    (UI Interaction)    Mouse
